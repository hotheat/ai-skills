# Position Mode

Use this mode for conceptual framework, agenda, perspective, manifesto, or position papers where the main contribution is an argument rather than a new artifact.

## Required Argument Chain

1. Central thesis: the claim the paper wants the field to accept.
2. Assumptions: explicit and implicit premises.
3. Argument chain: premise -> evidence -> conclusion.
4. Scope: where the argument applies and where it does not.
5. Counterarguments: plausible objections and whether the paper answers them.
6. Operational consequences: what decisions would change if the thesis is accepted.
7. Falsification: what evidence would weaken or overturn the thesis.

## Required Visual Modules

- Thesis Board: claim, stakes, strongest evidence, weakest premise.
- Argument Map: premises, evidence, conclusion, assumptions.
- Counterexample Cards: case, why it challenges the thesis, paper response.
- Decision Impact Matrix: stakeholder, changed action, risk.

## Conclusion Shape

Use this order:

1. Verdict on the thesis.
2. Strongest argument.
3. Weakest assumption.
4. Missing evidence that matters.
5. Practical implication.
